📌 Project Description

Big5 Safaris is a frontend web application built using modern web technologies. It focuses on delivering an engaging user experience for exploring safari destinations, wildlife conservation themes, and travel inspiration.

The platform highlights African safari culture through visuals, design, and structured navigation.

🚀 Features
🌍 Safari-themed homepage
🎨 Modern and responsive UI design
🖼️ High-quality wildlife and safari images
⚡ Fast performance using Vite
🧭 Client-side routing
📱 Mobile-friendly layout
🧑‍💻 Tech Stack
Frontend: React + TypeScript
Build Tool: Vite
Routing: React Router
Styling: CSS

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/fifi254-cyber/big5-safaris.git
cd big5-safaris
2. Install dependencies
npm install
3. Run the development server
npm run dev
4. Open in browser
http://localhost:5173
